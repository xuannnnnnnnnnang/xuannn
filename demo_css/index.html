TOMORROW X TOGETHER

简称TXT，意义为“互不相同的你我，为同一个梦想而相聚，携手共创明天”。官方粉丝名称 “MOA”，意Moments Of Alwaysness，永远的TOMORROW XTOGETHER和粉丝们在一起的每个瞬间, 是 BigHitEntertainment于2019年3月4日推出的韩国男子演唱组合，由崔秀彬 （SOOBIN）、崔然竣（YEONJUN）、崔机圭（BEOMGYU）、姜太显（TAEHYUN）、休宁凯（HUENINGKAI）5位成员组成，崔秀彬担任队长。

世界觀
